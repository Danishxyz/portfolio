# Portfolio Website

This is a modern, responsive personal portfolio website built with React, Tailwind CSS, Framer Motion, and GSAP.

## Getting Started

1.  **Install Dependencies** (if not already done):
    ```bash
    npm install
    ```

2.  **Run Development Server**:
    ```bash
    npm run dev
    ```

3.  **Build representing Production**:
    ```bash
    npm run build
    ```

## Project Structure

-   `src/components`: Reusable UI components (Navbar, etc.)
-   `src/sections`: Page sections (Hero, About, Skills, Projects, Education, Contact)
-   `tailwind.config.js`: Custom colors and font settings.

## customization

-   Edit `src/sections/` files to update your content.
-   Edit `src/index.css` or `tailwind.config.js` to change the theme.
